If you find the all edit scripts correctly, the buttons which show file numbers on the top turn green color.
The red color of the buttons on the top means that you miss or incorrectly make edit scripts.

After all of the buttons on the top turn green color, you can click the finish button and move to a last page.


After you finished generating edit scripts for the given code changes, you can submit your edit scripts by clicking "Finish" button.
Then you will see your code which can be used to obtain your reward in Amazon Mturk.

Before start, you practice how to create edit scripts on next page.
If you make edit scripts, you have to click "check!" button and confirm the codes on popup page.
The check button turns red or green according to whether edit scripts which you made create old code correctly.
After the check button turns green, the "start" button is turned on.
Then you click the "start" button and start our research project.
