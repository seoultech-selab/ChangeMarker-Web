Thank you for participating our research project on code changes.
In this project, we are investigating how code changes can be explained with primitive edit operations.

Once you start, you're given five code changes obtained from open source projects.
Each code change consists of two versions of source code.
Your task is generating an edit script -a series of edit operations -by selecting changed parts of the code.


There are four primitive edit operations you can use to describe a code change: insert, delete, move and update.
We will go through each edit operation with an example, and let you know how to generate the operations.
