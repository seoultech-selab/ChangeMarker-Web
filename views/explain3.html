The edit scripts are about how the codes change.
It have four types, "Insert", "Delete", "Move", "Update".

Example


After you find changes in codes, you drag that part and click the right mouse button.
Then, you can choose "Generate Delete", "Generate Move", etc.
If you click the "Generate ~", edit scripts are create on the "Edit Script" menu in the bottom.

Example

If you want to change or delete edit scripts which you made, click the delete button in "Option" column.
