Change Marker Tutorial

Thank you for participating our research project.
In this project, we're studying how various code changes can be decribed using simple types of edits.
More specifically, you are asked to describe code changes using Delete, Insert, Move and Update.

Your task is creating an edit script for each given code change.
An edit script consists of edit operations indicating edit types and changed code fragment.
With a quick tutorial, you will understand how you can create an edit script.
