Now you are ready to start the task of marking changes.
There will be five code changes you need to mark. Some changes are more complex than the others. Hence you need to carefully create your edit script for the changes.
After creating edit scripts for the given five changes, click "Finish" button to see your code for rewards.
You can copy the code and submit it to Amazon MTurk.
Please remind that we will check your edit scripts. If applying your edit script cannot convert the corresponding old version to the new version within reasonable range of differences, we may refuse to provide your rewards.
