Change Marker Basics

Here are the basics of Change Marker.
You can create an edit script for a given code change by marking changed parts of the code with Change Marker. In Change Marker, a code change is given like the following, similar to a typical compare editor.
 
 
To add an edit operation, drag a changed part of the code, then right click on it to see the menu.
You can see "Generate XXX" commands like the above example.
Then choose a command for an appropriate edit operation to describe how the selcted part was changed. For instance, you can click "Generate Delete", if you think the selected part was deleted.
 
To support creating edit scripts, Change Marker highlights line by line differences with red (deleted) and green (inserted). You can also click in the middle to jump to a changed part.
Remind that this is merely a simple support, and actual changed code do not need to be exactly matched to the highlighted part. Hence you need to examine a given change carefully, and describe it as you seem to fit using the commands.
